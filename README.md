# AthSys ver1
Enterprise Athletics Management System
Domain: appstore.co.ke
Developer: Mwamiri

## Features
- Athlete registration with optional World Athletics code
- Bib assignment and confrontation workflows
- Results capture via timing mats or manual entry
- Personal bests, season bests, team scoring
- Plugin ecosystem with tabbed management interface
- Auto backup, restoration, and self-healing
- Federation-compliant exports (Excel, HTML, XML/JSON)
- Cloud-native deployment with Docker/Coolify
