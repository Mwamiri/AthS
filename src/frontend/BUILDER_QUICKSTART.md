# üöÄ Page Builder - Quick Start (5 Minutes)

## Step 1: Access the Builder (30 seconds)
Open your browser and go to:
```
http://localhost:5000/builder
```

You'll see the **Builder Dashboard** with 4 main tools.

## Step 2: Create Your First Page (2 minutes)

1. Click **"Create New Page"** button
2. Fill in:
   - **Title**: "Welcome" 
   - **Slug**: "welcome"
   - **Status**: "Draft"
3. Click **"Create Page"** ‚Üí Redirects to Page Builder

## Step 3: Add Content (2 minutes)

### Add a Section
1. Click **"Sections"** tab (left sidebar)
2. Click **"+ Add Section"**
3. Choose section type: "Hero" (big banner section)
4. Click to add the section to your page

### Add Blocks to Section
1. Look at the **Library** tab (left sidebar)
2. See available components (buttons, cards, text, etc.)
3. **Drag** a "Text Block" from Library onto the section
4. **Drop** it to add the block

### Edit Block Content
1. **Click the block** you just added (middle canvas)
2. Right panel appears with properties
3. Edit the text content, colors, size
4. Changes save automatically!

## Step 4: Save Your Page (30 seconds)

- Click **"Save Draft"** to save without publishing
- OR click **"Publish"** to make it live

üéâ **Congratulations!** Your first page is live!

## Common Questions

### Q: How do I delete a block?
A: Click the block, then click the üóëÔ∏è (trash) button that appears

### Q: Can I undo changes?
A: Yes! Go to "Version History" to see previous versions and restore

### Q: How do I make text bigger?
A: Click the text block ‚Üí Use "Font Size" slider in right panel

### Q: How do I add links to buttons?
A: Click button block ‚Üí Set "Link URL" in properties ‚Üí Done!

### Q: Can I reuse designs?
A: Yes! Use the Component Library to save and reuse blocks

| Key | Action |
|-----|--------|
| `Delete` | Delete selected block |
| `Ctrl/Cmd + S` | Save draft |
| `Ctrl/Cmd + Z` | Undo (via version history) |
| `Esc` | Deselect block |

## Troubleshooting

**Problem**: "Component Library is empty"
- **Solution**: Go to Component Library tab ‚Üí Click "Create Component" to add some

**Problem**: "Theme colors not changing"
- **Solution**: Make sure the theme is activated (click checkmark in Theme Customizer)

**Problem**: "Changes not saving"
- **Solution**: Check browser console (F12) for errors, make sure you clicked Save/Publish
